    vue插件介绍

    李瑞波 2021/12/12

    # vue插件介绍

    # 1、插件与组件

    在讲解插件之前，我们先来了解一下插件与组件的关系。在我们的vue项目中使用组件的频率往往大于组件。

    在没有封装组件之前，如果不使用第三方插件，那么很多情况下我们会编写几个常用的组件来提供给页面使用，如Header、Input组件，而你可能需要在很多页面中引入并且通过components注册组件，但是像这样使用率很高的组件一般我们希望全局注册后直接就可以在相应页面使用，因此我们可以选择将他们封装成插件。

    # 2、插件的分类

    插件通常用来为 Vue 添加全局功能。插件的功能范围没有严格的限制——一般有下面几种：

    1. 添加全局方法或者 property。如：vue-custom-element (opens new window)
    2. 添加全局资源：指令/过滤器/过渡等。如 vue-touch (opens new window)
    3. 通过全局混入来添加一些组件选项。如 vue-router (opens new window)
    4. 添加 Vue 实例方法，通过把它们添加到 Vue.prototype 上实现。
    5. 一个库，提供自己的 API，同时提供上面提到的一个或多个功能。如 vue-router (opens new window)

    # 插件的开发与使用

    # 1、使用

    参数： {Object | Function } Plugin

    用法：安装Vue插件。如果插件是一个对象，则必须提供install方法。如果插件是一个函数，它会被作为install方法。调用install方法是，会将Vue作为参数传入。install方法被同一个插件多次调用时，插件也只会被安装一次。

    //main.js

import myPlugin from './plugins/index'

Vue.use(myPlugin)

    # 2、开发

    //plugins/index.js
export const install = function(Vue, options) {
	// 1、添加全局方法、property
  Vue.myGlobalMethod = function() {
   	...
  }

  Vue.myGlobalProperty = 'hello'

  // 2、添加全局资源
  Vue.directive('my-directive', {
    bind() {
      ...
    }
  })

  Vue.component('my-component', { /* ... */ })

  // 3、添加全局混入
  Vue.mixin({
    created(){
      ...
    }
  })
  // 4、添加Vue实例方法
  Vue.prototype.$myMethod = function(){
    ...
  }
}
export default {
	install
	...
}

    # Vue.use源码

    // scr/core/gloabl-api/use.js

export function initUse (Vue) {
  Vue.use = function (plugin) {
    const installedPlugins = (this._installedPlugins || (this._installedPlugins = []))
    if (installedPlugins.indexOf(plugin) > -1) {
      return this
    }

    // 其它参数
    const args = toArray(arguments, 1)
    args.unshift(this)
    if (typeof plugin.install === 'function') {
      plugin.install.apply(plugin, args)
    } else if (typeof plugin === 'function') {
      plugin.apply(null, args)
    }
    installedPlugins.push(plugin)
    return this
  }
}

export function toArray (list, start) {
  start = start || 0
  let i = list.length - start
  const ret = new Array(i)
  while (i--) {
    ret[i] = list[i + start]
  }
  return ret
}

    # 插件发布

    插件编写完后我们的目的除了本地引用注册外，可能更希望发布到线上供他人或其他项目使用，因此我们需要了解插件发布的方法。

    # 1、发布准备

    发布插件之前，你需要一个npm账号

    # 2、发布命令

    npm login

npm publish // 注意：不要直接使用 publish 命令，容易导致发布死循环；建议在package.json 中加一个 pub 命令， 里面调用 publish

    # 3、安装

    npm install my-plugin
